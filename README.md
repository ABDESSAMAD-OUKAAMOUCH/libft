**Libft** is a custom C library project from the **42 school curriculum**.  
It consists of reimplementing standard C library functions and creating additional utility functions that help in future projects.  

This project helps develop skills in:

- C programming fundamentals
- Pointers and memory management
- Strings and character handling
- Data structures (linked lists)
- Modular programming and code organization
- Writing reusable and testable code
